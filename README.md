## Step-1: Install Composer Dependencies:
composer install

## Step-2: Create a Copy of the Environment Configuration File:
cp .env.example .env

## Step-3: Generate an Application Key
php artisan key:generate

## Step-4: Run Database Migrations
php artisan migrate

## Step-6: Install VueJs
npm install, 
npm install vuex@next

## Step-6: Run npm
npm run dev
